# BSC Scan data service

**Sample ENV Config**

- Create .env file on root folder

```
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_SCHEMA_NAME=
PORT=
AWS_S3_ACCESS_KEY=
AWS_S3_SECRET_KEY=
AWS_S3_REGION=
AWS_S3_BUCKET=
AWS_S3_BUCKET_ENDPOINT=
```

# Before Running Project

- Please create database that have the same DB_SCHEMA_NAME before running below

# To run project

### Step 1

- Go to root directory

### Step 2

```
- go get
```

### Step 3

```
- go run main.go
```
