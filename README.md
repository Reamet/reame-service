# BSC Scan data service

**Sample ENV Config**

- Create .env file on root folder

```
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_SCHEMA_NAME=
```

# Before Running Project

- Please create database that have the same DB_SCHEMA_NAME before running below

# To run project

### Step 1

- Go to root directory

### Step 2

```
- go get
```

### Step 3

```
- go run main.go
```
